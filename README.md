## Projects

### [HealthDB](https://github.com/your-username/healthdb) — Distributed key-value store in Rust
Raft consensus from the ground up: leader election, log replication, and a write-ahead log with CRC32 integrity checks, all over TLS. A hands-on study of how consistency and durability actually get built rather than assumed.

### [BLE Mesh Sensor Network](https://github.com/your-username/ble-mesh-sensor-network) — Embedded-to-cloud pipeline
Senior design capstone. EFR32MG24 mesh nodes stream sensor telemetry over Bluetooth Mesh to a SiW917 Wi-Fi bridge, then up through MQTT to Firebase, with a React Native app for live monitoring and two-way device control. Full stack from silicon to phone.

### [What 2 Watch](https://what2watch.example.com) — Semantic movie recommendation engine
Go/Gin + SQLite backend, React frontend, and OpenAI embeddings for semantic search. Deployed on DigitalOcean behind Cloudflare and hardened with session cookies, rate limiting, and HMAC-signed requests. ([source](https://github.com/your-username/what2watch))
